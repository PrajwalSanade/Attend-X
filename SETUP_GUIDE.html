Face Recognition Attendance System - Quick Start

Step 1: Database Setup

First, set up your Supabase database:

  1. Go to your Supabase project
  2. Open SQL Editor
  3. Run the script: backend/FINAL_DATABASE_SETUP.sql

Step 2: Install Dependencies

Backend (Python):

cd backend
pip install -r requirements.txt

Frontend (Node.js):

cd frontend
npm install

Step 3: Start Development Servers

Option A: Start Everything at Once (Recommended)

# Windows
start-dev.bat

# Linux/Mac
./start-dev.sh

Option B: Start Manually

# Terminal 1 - Backend
cd backend
python start.py

# Terminal 2 - Frontend
cd frontend
npm run dev
